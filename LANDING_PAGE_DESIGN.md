# AIパース生成BOT ランディングページ設計

## 設計コンセプト
- **ターゲット**: 住宅営業マン、工務店、設計事務所
- **目的**: 3秒でサービス理解 → 30秒で友だち追加 → 即座に試せる
- **デザイン原則**: シンプル、信頼感、スピード重視

## デザインガイドライン

### カラーパレット
```
Primary:   #06C755 (LINE Green) - CTAボタン
Secondary: #00B900 (Dark Green) - アクセント
Background: #FFFFFF (White) - 清潔感
Text:      #333333 (Dark Gray) - 読みやすさ
Accent:    #F5F5F5 (Light Gray) - セクション背景
```

### タイポグラフィ
```
見出し(H1): 32-40px, Bold, Noto Sans JP
見出し(H2): 24-28px, Bold
本文:       16-18px, Regular
CTA:        18-20px, Bold
```

### レスポンシブデザイン
```
デスクトップ: 1200px max-width, 2カラム
タブレット:   768px, 1カラム
モバイル:     375px, 1カラム、タッチ最適化
```

### アニメーション
```
- スクロールイン: フェードイン + 上スライド
- Before/After: インタラクティブスライダー
- 生成デモ: 実際の使用フローのGIF
- ボタンホバー: 軽微なスケール + 影
```
